# رؤية عدالة (Adalah)

## ما هي عدالة؟

**عدالة** منصة ذكاء اصطناعي قانونية تهدف إلى جعل المعرفة القانونية والخدمات القانونية متاحة، دقيقة، وعادلة للجميع.

## الرسالة

تمكين المحامين والباحثين والمواطنين من الوصول إلى معلومات قانونية موثوقة عبر ذكاء اصطناعي يفهم السياق القانوني العربي، مع الحفاظ على أعلى معايير الدقة والأمان والشفافية.

## القيم الأساسية

| القيمة | المعنى |
|---|---|
| **العدالة** | إتاحة المعرفة القانونية دون تمييز |
| **الدقة** | كل إجابة مدعومة بمصدر قابل للتحقق |
| **الشفافية** | توضيح حدود النموذج ومصادره |
| **الأمان** | حماية بيانات المستخدمين والسرية المهنية |
| **الاستدامة** | بنية قابلة للتطوير لسنوات، لا لمشاريع مؤقتة |

## الفرق الجوهري

عدالة لا تعتمد على "Prompt طويل" في كل محادثة.  
بل تعتمد على **Engineering OS** — نظام تشغيل هندسي مدمج في المشروع:

```
طلب ميزة → تحليل → مقارنة حلول → اختيار → وثائق → تنفيذ → اختبار → مراجعة أمنية → تحديث وثائق → دمج
```

## الجمهور المستهدف

- **محامون ومكاتب محاماة** — بحث، صياغة، تحليل قضايا
- **باحثون قانونيون** — استكشاف التشريعات والسوابق
- **مواطنون** — فهم حقوقهم والإجراءات القانونية الأساسية
- **مطورون** — تكامل عبر API مع أنظمة قانونية أخرى

## مبادئ التصميم

1. **المعرفة في المشروع، لا في المحادثة** — كل سياق في `.docs/` و `.ai/`
2. **الفريق الافتراضي** — كل مهمة تمر على 8 أدوار متخصصة (راجع `.ai/Agents/`)
3. **القابلية للنقل** — الانتقال بين Cursor و Claude Code و GPT دون إعادة بناء العمليات
4. **القرارات موثقة** — ADR لكل قرار معماري مهم
