## قواعد التحديث

1. **كل قرار معماري** → ADR جديد في `ADR/`
2. **كل ميزة API** → تحديث `API/` قبل الدمج
3. **كل تغيير في قاعدة البيانات** → تحديث `Database/` + migration
4. **كل مراجعة أمنية** → تحديث `Security/` إن لزم
5. **كل تحسين أداء** → توثيق في `Performance/`
