## كيفية الاستخدام

1. **ميزة جديدة** → أضفها في `Features.md` → عيّنها لسبرنت
2. **خطأ** → أضفه في `Bugs.md` مع الخطورة
3. **بداية سبرنت** → أنشئ `Sprint-NNN.md`
4. **نهاية سبرنت** → حدّث `PROJECT_STATUS.md`

## التسمية

```
Sprint-NNN.md    → NNN رقم تسلسلي (001, 002, ...)
Features.md      → ميزة واحدة = سطر واحد مع الحالة
Bugs.md          → خطأ واحد = سطر واحد مع الخطورة والحالة
```
